# Telegram Dollar Price Bot

A Telegram bot that fetches and displays live USD/IRR exchange rates.

## Features
- /start — Welcome message
- /price — Get current dollar price in Iranian Rial

## Technologies
- Python
- python-telegram-bot
- REST API (tgju.org)

## How to Run
1. Install dependencies: pip install python-telegram-bot requests
2. Add your Telegram bot token to the TOKEN variable
3. Run: python bot.py
